O que é esse crime?

Por meio de dispositivos celulares ou redes de computadores conectados a uma rede de internet, criminosos fazem o uso disso para ludibriar, trapacear, enganar e sabotar a sociedade como um todo.

Normalmente são os hackers que utilizam técnicas avançadas ou até mesmo técnicas simples, mas que ainda funcionam.

Esses criminosos costumam roubar informações pessoais e dados financeiros.





Tipos de Crimes Cibernético

  • Fraudação pela Internet e por e-mail;
  • Falsificação de identidades, quando informações pessoais são usadas e roubadas;
  • Latrocínio de dados financeiros ou ligados a pagamento de cartões;
  • Comércio de dados corporativos;
  • Extorsão cibernética, que exige dinheiro para impedir o ataque ameaçado;
  • Um tipo de extorsão cibernética é Ataques de ransomware;
  • Quando criminosos exploram criptomoedas usando recursos que não possuem, isto é chamado de Cryptojacking.
  • Espionagem cibernética, quando hackers acessam dados de uma empresa ou do governo.


Casos populares de ataques

1. DDoS

Criminosos fazem extorsões cibernéticas e exigem dinheiro, paralisando um sistema ou uma rede. O DDoS consegue ser utilizado como tática de distração enquanto outro tipo de crime acontece.

2. Por malware

Dispositivos com Malware pode ser utilizado para diversos crimes cibernéticos. Possivelmente estará relacionado com destaque para usar um computador para realizar outros atos criminosos, roubar dados sigilosos ou causar danos ou dados.


3. Phishing

Uma expedição de phishing ocorre quando outras formas de comunicação ou e-mails de spam são enviadas em massa, com a intuito de induzir os destinatários a fazer algo que prejudique a segurança da organização em que trabalham ou a segurança deles. Operam com arquivos e links suspeitos, similar com o malware.

Um meio mais profissional é o “spear-phishing”, mensagens são criadas para aparentar com mensagens de uma fonte confiável. Por exemplo, para parecer que vieram diretamente do gerente de TI ou do CEO da empresa. Tem menos aparencia visual para identificação da falsidade.
